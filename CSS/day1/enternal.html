this is external

External CSS means writing CSS code in a separate .css file and linking it to the HTML file.

who to external

External CSS is used to style multiple web pages using one CSS file. helps keep HTML code clean and organized. Create a separate file with `.css` extension. Link the CSS file to HTML using the ` tag. It is best for large websites and easy maintenance.
